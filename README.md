🧑‍🤝‍🧑 KeenKeeper — Keep Your Friendships Alive

KeenKeeper is a modern friendship management web application designed to help users nurture meaningful relationships. It enables you to track interactions, monitor communication gaps, and maintain consistent connections with the people who matter most.

🚀 Overview

In today’s fast-paced world, staying in touch can be challenging. KeenKeeper solves this by providing a structured and intuitive platform to:

Track your communication history
Set relationship goals
Receive visual insights into your interaction patterns


✨ Core Features

👫 Friend Management
Create and manage detailed friend profiles
Track days since last interaction
Dynamic status indicators:
🔴 Overdue
🟡 Almost Due
🟢 On Track


⚡ Interaction Tracking
Log interactions instantly with:
📞 Call
💬 Text
🎥 Video
Automatically updates the timeline
Real-time toast notifications for user feedback


📜 Timeline System
Centralized view of all past interactions
Clean, chronological layout
Filter interactions by type for better clarity


📊 Friendship Analytics
Interactive pie chart representing communication patterns
Helps identify engagement trends
Built with modern data visualization tools


📱 Fully Responsive Design

Optimized for seamless performance across:

📱 Mobile devices
📟 Tablets
💻 Desktop screens


🛠️ Tech Stack
Technology	Purpose
⚛️ React.js	Frontend development
🌐 React Router DOM / App Router	Routing & navigation
🎨 Tailwind CSS + UI Library	Styling & responsiveness
📊 Recharts	Data visualization
🔔 React Toastify	Notifications


⚙️ Additional Highlights
