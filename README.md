# 🌱 KeenKeeper — Keep Your Friendships Alive

KeenKeeper is a smart friendship management web app that helps you nurture meaningful relationships by tracking interactions, monitoring communication gaps, and visualizing your social activity.

---

## 🚀 Features

### 👥 Friend Management

* Create and manage detailed friend profiles
* Track days since last interaction
* Smart status indicators (🟢 On Track, 🟡 Almost Due, 🔴 Overdue)

### 📞 Interaction Tracking

* Log interactions (Call, Text, Video)
* Real-time timeline updates
* Instant feedback with notifications

### 📊 Friendship Analytics

* Interactive charts (Pie Chart)

### 🕒 Timeline System

* Chronological interaction history
* Filter by interaction type (Call, Text, Video)
* Clean and intuitive UI

---

* ⚛️ React.js

* 🎨 Tailwind CSS + DaisyUI
